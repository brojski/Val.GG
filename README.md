<h2>Run Locally</h2>
<ol>
  <li>Move the folder <strong>'Val.GG'</strong> to C:\xampp\htdocs</li>
  <li>Run XAMPP Control Panel</li>
  <li>Start Apache</li>
  <li>Start MySQL</li>
  <li>Run your browser and type localhost/Val.GG<br /></li>
</ol>
